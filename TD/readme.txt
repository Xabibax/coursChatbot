Pré-requis pour les TP du cours de programmation de chatbots
------------------------------------------------------------
Les TP s'appuient sur les éléments suivants :
 - Node.js pour la programmation du webhook en Javascript
 - ngrok pour créer un tunnel réseau entre le server webhook et internet
 - un compte DialogFlow pour configurer l'agent conversationnel
 - un compte Slack pour utiliser une équipe Slack et sa messagerie comme canal de communication entre l'utilisateur et le bot
 
 1- Installer Node.js à partir du site https://nodejs.org/en/
 2- Installer Ngrok à partir du site https://ngrok.com/   - prendre la version gratuite - elle est largement suffisante pour le cours
 3- Créer un compte gratuit Dialogflow à partir du site https://dialogflow.com/
 4- Créer un compte Slack à partir du site https://slack.com/
 5- Créer un compte OpenWeatherMap pour interroger la météo d'une ville sur https://openweathermap.org/
 
 Les différents labs qui nécessite de rédiger un webhook en javascript sont fournis sous les répertoires labx
 fournis avec le readme. Il y a deux versions de chaque programme xx_squelette.js est le fichier de TD, l'autre
 fournit le code de correction.
